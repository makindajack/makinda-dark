# Changelog

All notable changes to the "Makinda Dark" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-27

### Added

- Complete theme overhaul with professionally curated colors
- 140+ token color scopes for comprehensive syntax highlighting
- 200+ UI color definitions for full editor coverage
- Semantic token colors support
- Brand color integration (Orange, Purple, Teal palette)
- Italic styling for comments, parameters, and keywords
- Documentation folder with installation, contributing, and publishing guides

### Changed

- Updated background to deeper black (#0e0e0f) for better contrast
- Enhanced function highlighting with vibrant orange (#ff8d37)
- Improved type/class colors with purple tones (#a78bfa)
- Refreshed string colors with teal (#2dd4bf)
- Better comment visibility with gray italic (#6b7280)
- Modernized UI elements with consistent color scheme

### Fixed

- Color contrast issues in various UI elements
- Terminal ANSI colors for better readability
- Git decoration colors for better visibility

## [0.2.6] - Previous Release

### Added

- Initial theme implementation
- Basic token colors
- Core UI theming

## [0.2.0] - Early Development

### Added

- First public release
- Basic dark theme structure
